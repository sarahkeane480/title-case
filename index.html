Below is an example of how you can use JavaScript to process text input by the user and display it reformatted as Title Case. For example: If the user enters "hotdog" it should output "Hotdog" (and suprise: it does !!!!)

To do this, I created a form element that contains two <input> elements. One is type="text", which creates a text input for the user to enter text into. The other one is of type="submit", which creates a submit button which sends all the form data upon activation.

Then, I created a JavaScript function named toTitleCase(str) that takes a string as an argument, and returns it reformatted to be Title case.

To do this, it first extracts the first character of the string using str.charAt(0), and then use the .toUpperCase method to make it uppercase. After chaining, it becomes str.charAt(0).toUpperCase().

Then, I use str.slice(1) to extract all of the characters besides the first character. The reason that works is because if only one argument is passed to .slice(), it takes the character at that index, followed by all other characters past it. Then we just chain the .toLowerCase() method to the end, and end up with str.slice(1).toLowerCase()!

By the end, the function looks like this:

  function toTitleCase(str) {
    // Upper case first letter + Lower case other letters
    return str.charAt(0).toUpperCase() + str.slice(1).toLowerCase();
  }

After that, I added an event listener to the type="submit" button by using the .addEventListener() method on the submit button's element, which I acquired using document.querySelector("#submit") (by the way, submit is the ID of the submit button... who could've guessed!)

FINALLY, all I had left to do was create an anonymous function to pass into .addEventListener() that would overwrite the .innerHTML property of the div containing the text output, obtained with document.querySelector("#output")... and finally, we're all set!

...Oh wait, no thats not all, I also had to make the submit button not do its normal stuff. Obviously!! (Just kidding, this is one of the more confusing pieces to remember, and you don't want to forget it!). To do that, all I had to do is add ev.preventDefault() to the start of the function, preventing the submit button's event ev from doing its normal thing. Fantastic!

And now, FINALLY, for real this time, it's done! Here is the final code of the event listener:

  mySubmit.addEventListener("click", function(ev){
    ev.preventDefault(); // Make the "click" event not do what it normally does
    const myInput = document.querySelector("#input");
    const myOutput = document.querySelector("#output");
    myOutput.innerHTML = toTitleCase(myInput.value);
    // alert(myInput.value)
  });
